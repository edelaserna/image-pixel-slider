# Image Pixel Slider Web Component
This is an interactive microscopy image pixel size slider to compliment lessons on sampling in digital imaging.

## How to embed in a Moodle page
1. Edit the Moodle page contents.
2. Go to View>Source Code.
3. Paste the following code where you would like the slider to appear: 
```html
<div data-component="image-pixel-slider">Â </div>
<p>
  <script type="module"
    src="https://edelaserna.github.io/image-pixel-slider/"
    crossorigin="anonymous"></script>
</p>
```
4. Save the source code.
5. Save page.
